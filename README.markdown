# mongo-chat
Mongo chat is a sample Node.js chat application that uses socket.io and mongoose. 

## Dependencies

- node.js
- mongodb
- npm

## Installation

    git clone [repo]
    cd mongo-chat
    npm install

## Running the app

1. Make sure mongodb is running
`mongod`

2. Run the app
`node app.js`
